Peixoto's patch (https://www.patreon.com/user?u=44312848&fan_landing=true)
quick readme

This program contains a collection of patches for old windows games, increasing 
their compatibility with newer windows versions, fixing bugs, glitches, poor 
frame rates and also improving the games with things like forced resolution, 
widescreen and modding tools

It is also possible to create your own patches with the program's user interface 
which allows you to create new game profiles and cofigure several tweaks for 
each profile

More detailed documentation can be viewed clicking on Peixoto.bat - it launches
program\qt\HelpQt.exe which is simply autohotkey (https://autohotkey.com) and 
runs the script program\qt\HelpQt.ahk which you can inspect with any text editor
 
Dependencies:
- Microsoft Visual C++ 2010 Redistributable (x86)
- Microsoft Visual C++ 2015 Redistributable (x86)
- DirectX End-User Runtimes (June 2010 or latter)
